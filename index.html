Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is used as the framing or scafolding for a page.

CSS

  • CSS stands for Cascading Style Sheets, and is the language used to add style and PIZAZZ to your pages.
  • It's written in a style that looks a lot like a JSON object, which is a javascript file type that's a collection of items with varying types of attributes.
  • It helps with organization by taking the aesthetic load out of your HTML document.

git

  • git is the language used with gitbash and guthub for version control.
  • It allows multiple people to work in a single workspace/repository while not changing the main branch or stopping others from doing their work.
  • It also helps revert back to previous versions if ever a bug comes up that can't be fixed easily.
  • git status: checks what branch we are currently on
  • git checkout -b "branch-name": creates a new branch and switches to it
  • pwd (print working directory): shows you which directory you're currently in
  • ls: shows you what folders are available in your current directory
  • cd 'new directory name': lets you change your current directory to whatever directory you choose (don't include quotations)
  • cd ..: exits your current directory and places you in the directory immediately before your current one

JavaScript

  • JavaScript is the language that adds functionality to webpages, and it able to gather and use data and input from users.
  • It allows the developer to do things that aren't possible in HTML or CSS.
